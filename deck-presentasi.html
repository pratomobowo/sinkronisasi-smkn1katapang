PROGRAM 3

Git & DevOps Pipeline

Zero Git — fundamental daily driver — ada di >90% job description developer

Solusi

  • Kelas X: Git dasar (init, add, commit, push)
  • Kelas XI: Branching, merge, PR, code review
  • Kelas XII: CI/CD + GitHub Actions + Docker prod

Kenapa Penting

  • 450+ lowongan Git jobs di Indonesia
  • 56% developer pake container
  • 100% gratis — GitHub, Docker

Zero Cost — High Impact: Syarat mutlak kerja, bisa mulai dari pertemuan pertama

PROGRAM 4

Database Modern

SQL manual kuat tapi zero ORM & NoSQL

Solusi

  • Kelas XI: SQL manual tetap (fondasi) + Prisma ORM
  • Kelas XII: Supabase/Firebase — NoSQL, real-time, auth

Prinsip

SQL manual tetap diajar — ORM di atas SQL, bukan ganti SQL. Fondasi SQL tetap fundamental.

Yang Tidak Berubah: SQL manual tetap fondasi. ORM hanya tools — kalau SQL lemah, ORM jadi masalah.

VISION

Ke Mana Arah Industri 2027-2030?

📅 2027-2028 (Jangka Pendek)

  • AI coding assistant jadi default
  • AI-augmented testing — TDD kebiasaan
  • API-first arsitektur default
  • Cybersecurity syarat dasar (OWASP)

📅 2029-2030 (Jangka Menengah)

  • AI-Human Collaborative Engineering
  • AI generate 70-80% kode — skill review jadi pembeda
  • Peran baru: Prompt Engineer, AI QA Engineer
  • AI-native apps — AI sebagai core, bukan add-on

✅ Yang PASTI Tidak Berubah

  • Logika & algoritma — AI bisa generate, gak bisa ganti problem solving
  • SQL — ORM wrapper, SQL tetap fundamental
  • HTML/CSS/JS — web tidak kemana-mana
  • Git — kolaborasi tanpa version control tidak mungkin
  • Soft skills — komunikasi, kolaborasi, critical thinking
