PROGRAM 4

Database Modern

SQL manual kuat tapi zero ORM & NoSQL

📚 Solusi

  • Kelas XI: SQL manual tetap (fondasi) + Prisma ORM
  • Kelas XII: Supabase/Firebase (NoSQL, real-time, auth)

⭐ Prinsip

SQL manual tetap diajar — ORM di atas SQL, bukan ganti SQL. Fondasi SQL tetap fundamental, ORM hanya mempermudah development modern.

YANG TIDAK BERUBAH

SQL manual tetap jadi fondasi. Lulusan harus paham query raw, join, aggregate, subquery. ORM hanya tools — kalau SQL lemah, ORM jadi masalah.

VISION

Ke Mana Arah Industri 2027-2030?

📅 2027-2028 (Jangka Pendek)

  • AI coding assistant jadi default — bukan tools tambahan
  • AI-augmented testing — TDD jadi kebiasaan
  • API-first & Headless Architecture — arsitektur default
  • Cybersecurity jadi syarat dasar (OWASP Top 10)

📅 2029-2030 (Jangka Menengah)

  • AI-Human Collaborative Engineering — programmer sebagai "orchestrator"
  • AI generate 70-80% kode — skill baca, review, debug jadi pembeda
  • Peran baru: Prompt Engineer, AI QA Engineer, AI-Assisted Developer
  • AI-native apps — AI sebagai core, bukan add-on

✅ Yang PASTI Tidak Berubah

  • Logika & algoritma — AI bisa generate, gak bisa ganti problem solving
  • SQL — ORM wrapper, SQL tetap fundamental
  • HTML/CSS/JS — web tidak kemana-mana
  • Git — kolaborasi tidak mungkin tanpa version control
  • Soft skills — komunikasi, kolaborasi, critical thinking

FORMAT

Format Penyelarasan — Siap Diisi Bareng

📋 Dokumen yang Dibawa

  • Format standar penyelarasan kurikulum SMK dengan IDUKA
  • 18 baris item gap — tinggal diisi rekomendasi dan prioritas
  • Kolom: mata pelajaran → CP existing → kompetensi industri → gap → rekomendasi perubahan
  • Siap diisi bersama tim sekolah
  • Bisa langsung diisi hari ini atau dibawa pulang untuk didiskusikan

📦 Dokumen Pendukung (Fisik)

  • Format penyelarasan kosong — Cetak A4 (5-10 lembar)
  • Slide deck ini — Cetak / PDF
  • Summary gap analysis — 1 halaman A4
  • Profil Next IT — 1 halaman
  • 1-pager data kunci — Infografis data utama

Format ini siap diisi bareng di meeting hari ini — tinggal diskusi kolom per kolom, kesepakatan program mana yang dijalankan duluan.
